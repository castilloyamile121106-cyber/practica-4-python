# Práctica 4 Python

Este repositorio contiene un generador de números aleatorios utilizando el método **Congruencial Lineal**.

## Estructura del Proyecto
* [generadornumaleatorios.py](file:///C:/Users/Zaida%20Yamile/Practica%204%20python/generadornumaleatorios.py): Script en Python que implementa el generador congruencial lineal.

## Parámetros del Generador
* **Módulo ($m$)**: 4096
* **Multiplicador ($a$)**: 5
* **Incremento ($c$)**: 1
* **Semilla ($X_0$)**: 1234 (Modificable en el script)

## Ejecución
Para correr el script en tu computadora, ejecuta el siguiente comando:
```bash
python generadornumaleatorios.py
```
El script generará un archivo llamado `resultados_generador.txt` en el mismo directorio con el período completo de números generados.
